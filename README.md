# js 구성<br>
js폴더는 6개의 파일로 구성되어있습니다.<br>
1) 배경화면 랜덤 background.js<br>
2) 시간을 보여주는 clock.js<br>
3) 사용자의 아이디를 보여주는 greeting.js<br>
4) 투두 리스트 <br>
5) 명언 랜덤 quotes.js<br>
6) 사용자 위치와, 날씨 및 날씨 아이콘을 보여주는 weather.js<br>
