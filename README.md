## js 구성
js폴더는 6개의 파일로 구성되어있습니다.
1) 배경화면 랜덤 background.js
2) 시간을 보여주는 clock.js
3) 사용자의 아이디를 보여주는 greeting.js
4) 명언 랜덤 quotes.js
5) 사용자 위치와, 날씨 및 날씨 아이콘을 보여주는 weather.js
